**This guide will help you understand how to set up a basic monitoring system using:**

PostgreSQL (stores your data)
Telegraf (collects metrics from PostgreSQL)
InfluxDB (stores those metrics)
Grafana (visualizes the data as graphs)

**What You'll Do**

Create a sample PostgreSQL table.
Configure Telegraf to fetch the table's data.
Send that data to InfluxDB.
Visualize the data in Grafana.

**Tools You'll Need**

Ubuntu/Linux terminal
PostgreSQL
Telegraf
InfluxDB
Grafana (all free and open-source)
